# Instructions for Candidate

1. Clone the repository.
2. Load `SampleData/SampleData.json` in a simple repository/service (or use in-memory).
3. Implement `TeacherController` with action `Profile(int id)` that returns teacher profile page.
4. Use a ViewModel for the page.
5. Use Bootstrap 3 or 5 for layout. The page should be responsive.
6. Commit your work and push to your fork/branch or create a pull request.
